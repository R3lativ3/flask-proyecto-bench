Acerca del Proyecto Bench

Sistema de gestión de desarrolladores core y bench para proyectos.

¿Qué es el Proyecto Bench?

El Proyecto Bench es un sistema diseñado para gestionar de manera eficiente las asignaciones de desarrolladores en diferentes proyectos, permitiendo la rotación entre roles "Core" y "Bench" según las necesidades del negocio.

Características Principales

  • Gestión de proyectos y asignaciones
  • Sistema de solicitudes de bench y días libres
  • Calendario de asignaciones dinámico
  • Roles y permisos de usuario
  • Dashboard con métricas en tiempo real

Tecnologías Utilizadas

Backend
  • Python 3.9+
  • Flask
  • SQLAlchemy
  • Flask-Login
  • WTForms
Frontend
  • HTML5 & CSS3
  • Bootstrap 5
  • JavaScript (ES6+)
  • Jinja2 Templates
